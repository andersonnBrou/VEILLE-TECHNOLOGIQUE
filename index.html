LES TECHNOLOGIES FRONT-END Vs BACK-END

Le front-end et le back-end sont deux notions clés du domaine de la tech.

Ces deux types de developpement web interviennent dans la creations des applications,

des sites web et de tous produits dématérialisé qui comporte une interface graphique.

Les technologies back-end

Le back est une partie importe du développement,sans laquelle un site web

ou une application ne pourrait marcher.

Meme s'il est indispensable,le back-end est un

ensemble d'opérations qui ne sont pas visibles pour les visiteurs.

le back-end est hiérarchisé en trois partie

le serveur ou hébergeur
l'application
La base

les developpeurs back-end utilise des les langages de programation comme: Python - PHP - Ruby pour mettre en place et configurer le serveur.

Les technologies front-end

Ils désignent les élements d'un site ou d'une application que les utilisateurs voient à l'ecran et avec lesquels il vont interagir.

Les langages de programation HTML,CSS,JavaScrip sont des combinaisons interpretétés par le navigateur et percu par les internautes.

En général, le front-end se compose d'un design et d'un code HTML,CSS, JavaScrip et jQuery. Le design est crée par le web designer qui vas realiser des maquettes graphiques avec des outils dédies, comme photoshop ou Fireworks.
